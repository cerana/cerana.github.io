Cerana

Computing Construction Kit

What Is Cerana?

Cerana makes using a cluster of computing nodes for any task quick and efficient. It combines a purpose built Linux distribution with Go based server orchestration tools.

How Does Cerana Work?

Cerana uses a custom Linux distribution built with clustered computing in mind. With CeranaOS you get delegatable ZFS filesystems, secure contained processes with network isolation, and efficient virtualization.

To complete the system, we have API based orchestration to manage datasets and processes within your cluster. Our Go based services include clustered support that automagically uses your infrastructure efficiently as it grows.

For more information, be sure to check out our documentation.

Who Is Behind Cerana?

Cerana is primarily developed by engineers at OmniTI, with contributions from various members of the open source community. If you are into Linux kernel hacking and want to improve the state of Linux containers or ZFS on Linux, OmniTI is hiring.

Stay Involved

As a new open source project, we welcome contributors from all areas; OS hackers, Go devs, documentation folks, website help; all are welcome! If you are interested in testing out Cerana, please join us on freenode on #cerana.
